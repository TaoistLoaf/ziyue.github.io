👉 如果你的 Hugo 站点还没有多语言支持 检查 config.toml 里是否有 [languages] 配置，如果没有，就添加（参考上面）。 创建 i18n/translations.yaml，让 Hugo 知道 translations 这个键的翻译。 确保 content/zh/ 和 content/en/ 里有文章，否则 .Translations 是空的，不会显示切换选项。 👉 如果你已经启用了 Hugo 多语言，但切换按钮没有显示 检查 content/zh/ 和 content/en/ 是否有对应的翻译文章。 尝试在 Hugo 启动时强制多语言模式： sh Copy Edit hugo server –disableFastRender –printI18nWarnings 这样可以检查 Hugo 是否正确识别多语言内容。 手动添加一个语言切换按钮（如果 .Translations 为空）： html Copy Edit
